# ExchangeRatesApp

The application uses API NBP (http://api.nbp.pl/) for currency exchange rates.

Input data:
- currency code (EUR);
- start date (RRRR-MM-DD);
- end date (RRRR-MM-DD).

Output data:
- average buying rate;
- standard deviation of the sale prices.

# Built With
- Java - Logic
- JavaFX - View
- Maven - Dependency Management
